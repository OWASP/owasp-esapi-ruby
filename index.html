The Owasp Esapi Ruby is a port for outstanding release quality Owasp Esapi project to the Ruby programming language.
The idea is to build a Ruby gem containing the Esapi concepts implemented in Ruby classes so people using Ruby in their Rails application can have security into them.
From here you can take a look at the code, you can join the mailing list or give a look to the project roadmap.
